## React boilerplate

#### Features

- Webpack 4
- Styled components support
- Jest and Enzyme for testing
- Cosmos 5
- Eslint configuration

#### Getting started

1. Clone the repo
   `$ git clone <repo-url> my-awesome-project`
2. Install dependencies
   `$ npm install`
3. Start
   `$ npm start`
4. Run tests
   `$ npm test`
5. Run Cosmos
   `$ npm run cosmos`
